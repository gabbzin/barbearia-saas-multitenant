## Requisitos técnicos

- Use a Vercel AI SDK e a rota "api/chat" para construir o chat.
- Ignore o botão de reservar
- Troque o Agenda.AI por Aparatus
- **APENAS** envie e receba as mensagens, não efetue nenhum tipo de formatação (vamos usar Streamdown depois).
- Use o Context7 para buscar na documentação da Vercel AI sdk
- O botão de gravar áudio não deve fazer nada por enquanto
- Adicione ao lado do botão de gravar áudio um botão de enviar mensagem
- Deve ser possível enviar a mensagem ao apertar enter
